# nodejs-simple-server

## Description 
<p>nodejs-simple-server is a Node.js application using the built-in 'http' module to create a simple HTTP server. It serves a static HTML file as a response and logs the request with a timestamp. Also, npm package nodemon is used for live reloading during development. This setup allows for quick iteration and testing of the server without the need for manual restarts.</p>

## Features
- Node.js
- npm package nodemon
- HTTP module
- HTML file
- Timestamp logging
- Static file serving
- Live reloading during development

### Start the app
```shell
npm install
```
or
```shell
yarn or yarn install
```
then
```shell
npm run server
```
or

```shell
yarn server
```

### Run production mode
```shell
npm run prod
```
or

```shell
yarn prod
```
